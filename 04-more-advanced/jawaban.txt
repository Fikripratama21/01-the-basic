1. git revert sebagai alat untuk membatalkan perubahan yang telah dilakukan sementara git reset adalah untuk membatalkan perubahan yang belum dilakukan
2. git merge adalah perintah untuk menggabungkan cabang git, git rebase adalah proses memindahkan atau menggabungkan urutan commit ke new base commit
3. git stash pop adalah perintah yang menghapus stash (paling atas, secara default) saat menerapkannya
sedangkan git stash apply meninggalkannya di dalam stash list yang kemungkinan digunakan untuk nanti.
